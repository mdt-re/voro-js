# voro-js

An [Emscripten](https://emscripten.org/) implementation of [voro++](https://github.com/chr1shr/voro) for Javascript & WebAssembly.





## Usage with npm

For local use run
 * npm run build
 * npm link
inside the main directory and then link in the project with
 * npm link @mdt-re/voro-js
