# voro-js Documentation

Welcome to the documentation for `voro-js`, a TypeScript & WebAssembly implementation of the Voro++ library. This library allows for the dynamic generation of 3D Voronoi tessellations directly in the browser or Node.js environment.

## Table of Contents

1.  **[Installation](./installation.md)**
    How to install the package via npm or yarn.

2.  **[Getting Started](./getting_started.md)**
    A step-by-step guide to your first Voronoi simulation.

3.  **[API Reference](./api.md)**
    Detailed information about the main classes `VoronoiContext3D` and `VoronoiCell3D`.

4.  **[Memory Management](./memory.md)**
    **Crucial**: Learn how to manage the lifecycle of WebAssembly objects to prevent memory leaks.

5.  **[Advanced Usage](./advanced.md)**
    Guides on Voronoi relaxation, custom boundaries, and performance tips.
