# Installation

To start using `voro-js` in your project, you need to install the package via your preferred package manager.

## Package Managers

### npm

```bash
npm install voro-js
```

### Yarn

```bash
yarn add voro-js
```

## Environment Support

`voro-js` is compiled to WebAssembly and supports the following environments:

*   **Node.js**: Version 12 or higher.
*   **Browsers**: Modern browsers with WebAssembly support (Chrome, Firefox, Safari, Edge).
